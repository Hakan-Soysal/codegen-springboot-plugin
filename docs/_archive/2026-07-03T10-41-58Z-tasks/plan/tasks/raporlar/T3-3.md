   tekil `{module}/Errors.java` (aggregate, sabitler için sorun yok) — bu ayrım task metninde de

   parantez-içi talimatının yorumlanması.** Task Step 5.4 şu ifadeyi taşıyor: "(gövde `throw new
   UnsupportedOperationException("outbox: doldurulacak-degil — altyapı stub")` DEĞİL: .NET paritesi
   = NotImplemented stub; yorumla işaretle)". Bu, .NET'in mesajındaki "doldurulacak" kelimesinin

   (UnsupportedOperationException fırlatma; .NET NotImplementedException paritesi) korunmuş oldu.

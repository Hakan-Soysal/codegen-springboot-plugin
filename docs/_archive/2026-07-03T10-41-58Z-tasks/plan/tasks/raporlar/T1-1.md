`JsonMappingException("unknown ExprNode shape")`.

### 5.5 — Unit testler
`gen-core/src/test/java/techgen/core/model/ExprNodeTest.java` — **10 test** (istenen ≥8):
1. cmp/path/number direkt JSON'dan parse
2. fixture'dan (`operations[0].validation[0].ast`) parse → 6.2 acceptance ile birebir aynı
3. nested and/or parse
4. AggNode parse
5. CallNode parse
6. DurationNode parse
7. round-trip (parse→serialize→parse) — **record equality** ile karşılaştırma (`assertEquals`,
   string karşılaştırması DEĞİL)
8. bilinmeyen literal kind (`{"kind":"weird","value":1}`) → `assertThrows(JsonMappingException)`
9. bilinmeyen şekil (`{"foo":1}`) → `assertThrows(JsonMappingException)`
10. **bilinmeyen nodeKind toleransı** pozitif test: `{"node":"xor",...}` → gerçekten
    `BinaryNode("xor", null, PathNode(a), PathNode(b))` üretildiği `assertEquals` ile kanıtlandı
    (yalnızca "patlamıyor" değil, üretilen değer de doğrulandı).
